
# Client_docker
Interface React qui permet d’interagir avec les différents services et de tester les fonctionnalités du projet. Il s’agit d’une preuve de concept pour le modèle ainsi que d’une démonstration des capacités des différents services d’étiquetage.

## Services de l'interface

https://github.com/WilliamYn/learning-captioning-model

https://github.com/midoxc/nhl_image_classifier
